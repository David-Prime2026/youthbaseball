# Premier Select - Baseball Player KPI Tracking

A comprehensive youth baseball player performance tracking application built with React, TypeScript, and Tailwind CSS.

## Features

- **Player Management**: Track multiple players with profiles, photos, and age groups
- **Performance Metrics**: Monitor batting, pitching, running, and strength stats
- **Game Stats Import**: CSV import with automatic player profile creation
- **Advanced Analytics**: YoY comparisons, outlier detection, AI insights
- **Offline Sync**: Queue data when offline, sync when connected
- **Visual Charts**: Performance trends with Recharts
- **Season Tracking**: Spring, Summer, Fall, and Year-to-Date views

## Tech Stack

- React 18.3
- TypeScript
- Tailwind CSS v4
- Vite
- Recharts
- React Router
- Radix UI Components

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
